Array Practice - Companies

Question

Create an array to store companies:

Bloomberg, Microsoft, Uber, Google, IBM, Netflix

Solution

let companies = ["Bloomberg", "Microsoft", "Uber", "Google", "IBM", "Netflix"]; // a. Remove first company companies.shift(); // b. Remove Uber & add Ola companies.splice(1, 1, "Ola"); // c. Add Amazon at the end companies.push("Amazon"); console.log(companies);
Final Output: ["Microsoft", "Ola", "Google", "IBM", "Netflix", "Amazon"]
