Array Methods in JavaScript

Slice( )

Definition: returns a piece of the array

let arr = [10, 20, 30, 40, 50]; let result = arr.slice(1, 4); console.log(result);
Output: [20, 30, 40]

Splice( )

Definition: change original array

let arr = [10, 20, 30, 40, 50]; arr.splice(2, 2, 99, 100); console.log(arr);
Output: [10, 20, 99, 100, 50]
