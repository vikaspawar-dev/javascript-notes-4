Array

  • Concat( ) : joins multiple arrays & returns result let arr1 = [1, 2]; let arr2 = [3, 4]; let result = arr1.concat(arr2); // [1, 2, 3, 4]
  • Unshift( ) : add to start let arr = [2, 3]; arr.unshift(1); // [1, 2, 3]
  • Shift( ) : delete from start & return let arr = [1, 2, 3]; arr.shift(); // [2, 3]
