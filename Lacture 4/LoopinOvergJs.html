Looping over an Array

Print all elements of an array

let arr = ["A", "B", "C", "D"];

for (let i = 0; i < arr.length; i++) {
    console.log(arr[i]);
}

Explaine:

📦 Code:
let arr = ["A", "B", "C", "D"];

for (let i = 0; i < arr.length; i++) {
    console.log(arr[i]);
}
🔍 Step by Step:
1️⃣ Array
arr = ["A", "B", "C", "D"];

👉 Isme 4 elements hai
👉 Index: 0, 1, 2, 3

2️⃣ Loop start
let i = 0;

👉 Start index 0 se

3️⃣ Condition
i < arr.length

👉 arr.length = 4
👉 Loop chalega jab tak i < 4

4️⃣ Print
console.log(arr[i]);

👉 Har baar current index ka element print hoga

🔄 Loop ka flow:
i	arr[i]	Output
0	A	A
1	B	B
2	C	C
3	D	D
